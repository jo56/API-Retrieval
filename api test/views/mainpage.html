Coronavirus Data Retrieval

This application takes data from a free api that tracks coronavirus data for countries around the world, featuring the most recent cases/deaths/testing updates

To use, simply press the "Fetch Data" button to call the information from the api.
After receiving this information, enter text into the search bar and hit "search" to filter the results, matching any country that starts with the characters you entered
